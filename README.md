Environmental_bot adalah chatbot discord buatan saya (Max) untuk membantu remaja-remaja yang mengalami kesulitan menjaga kebersihan dan mengurangi sampah dirumah mereka. 
Chatbot ini dibuat sebagai tugas M2L2 python pro di kodland yang dimana tema tugasnya adalah membuat chatbot dicord yang membantu orang untuk memilah sampah
Cara penggunaan:
1)Masukan server id ke line 16    channel = bot.get_channel(#masukan id server disini)
*Aktifkan Developer Mode di pengaturan pengguna, lalu klik kanan ikon server di daftar server dan pilih Copy ID/Salin ID.
2)jalankan/run bot
3)ada 2 command yang anda bisa gunakan:
- $commands (untuk menunjukan commands yang bisa dipakai)
- $tips (untuk memberikan informasi mengenai pengurangan sampah)
